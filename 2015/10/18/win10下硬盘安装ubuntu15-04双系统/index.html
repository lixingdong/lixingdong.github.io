win10下安装ubuntu15.04双系统

发表于 2015-10-18   |  

笔者之前没有安装过ubuntu系统，属于是安装系统界的小白，今天把在win10下通过硬盘安装ubuntu15.04系统的安装过程记录于此，用以备忘

物料

  • win10系统（是在win10下装ubuntu）
  • ubuntu Desktop 15.04 iso文件，请移步 官网) 下载
  • easyBCD 2.2工具（免费软件，用来添加系统启动项，下载链接）
  • 一款能够解压iso文件的解压软件(笔者使用的是7zip)

分区

为了安装双系统，我们需要在win10下给ubuntu系统分出一块新的分区，在win10桌面左下角的windows logo上右键点击，选择磁盘管理,选择一块空间比较大的分区（笔者这里在C盘上分区），在上面右键点击压缩卷，然后输入想要给ubuntu系统的空间大小。如下图，笔者给ubuntu分配了10G空间，因为当时的截图只是单纯为了写这篇博客，所以分配空间分的很少，建议读者分配20G空间左右。分区颜色为黑色时表示分区成功。

读者此时需要留意图片中的标注，后面配置ubuntu的iso文件位置时用的到。

启动项

添加启动项

安装easyBCD软件，启动之后点击添加新条目,之后选择NeoGrub启动项，在之后点击添加，然后点击配置按钮，如下图：

配置启动项

点击上面配置按钮之后，会弹出来一个menu.list文档，将文档内容删除，把如下内容拷贝到文档中，保存.

title Install Ubuntu

root (hd0,1)

kernel (hd0,1)/vmlinuz.efi boot=casper iso-scan/filename=/ubuntu-15.04-desktop-amd64.iso locale=zh_CN.UTF-8

initrd (hd0,1)/initrd.lz

title reboot

reboot

title halt

halt

hd(0,1)中的第一个0代表是在ubuntu iso在本机中的哪一个物理硬盘上，因为大部分电脑都只有一个硬盘，所以此处的0可以不变，后面的1代表我们的iso在该硬盘中的第几个分区中，关于分区可以看第一张图片上面的标注内容。这里我们把ubuntu 的iso文件放在C盘根目录下，所以填写的是hd(0,1)
第三行中filename后面的是你下载的iso文件的全名，注意，这个名字不能错哦
从上面的代码中可以看出，我们还需要把initrd.lz文件和vmlinuz.efi文件同样拷贝到C盘根目录下，这两个文件在安装ubuntu的过程中用得到，使用解压缩问价打开ubuntu的iso文件，在casper目录下找到这两个文件，拷贝出来就可以。至此，安装ubuntu的前期准备工作就完成了。最终效果如下图:

重启系统

保存了上面的menu.list文件之后，重启系统，选择NeoGrub引导加载器的启动方式。

Install Ubuntu

之后，就应该出现如下图的选项。，选择Install Ubuntu选项

安装ing

在之后就应该刷刷的出现满屏幕的文字，如下图，如果没有出现，则说明之前的menu.list配置的不对，这时只要重新进入win10下，重新配置menu.list文件。此处可能出错的地方就是读者可能没有看清楚自己的C盘到底是物理硬盘上的第几个分区。还是仔细看第一张图中的标注。

卸载iso文件

在上面一步中经过较长时间的等待（笔者的固态硬盘速度杠杠的，半分钟就结束了:)）,此时应该进入了一个小系统中，如下图，此时千万不要点击”安装Ubuntu图标!!”

因为我们打算给ubuntu重新分区，所以需要把已经挂载进来的iso文件卸载，这样才能重新分区。按住CTRL + ALT + T键，打开命令行终端，输入如下指令：

sudo umount -l /isodevice

这个是硬盘安装系统时候需要运行的命令。是把已经挂在的iso文件卸载。因为iso文件是存储在硬盘上的，所以只有卸载之后才能够给该硬盘分区。显然，如果你不是用硬盘安装系统就不用这条命令。

注意上面的命令中的空格和小写的L。如下图：

分区

下面就要进入到分区的过程中了，此时界面上应该有一个名字叫做空闲的分区，这就是之前在win10系统中为安装ubuntu新建的分区。

交换空间分区

双击”空闲”分区，首先给”交换空间”分区，一般来讲，交换空间的大小应该设置为物理内存的2倍，笔者此文旨在备忘，所以就只分配了1G大小。如下图：

“/“分区 “/boot”分区 和 “/home”分区

建议的大小如下： / 10G ， /boot 200M ，/home 分区大小为剩余所有的空间，最终效果如下图：

重启系统

安装结束之后，应该会出现如下图：选择重启就OK了。

删除启动项和iso文件

安装完成之后，读者可以进入到win10系统下，使用easyBCD软件把之前新建的”NeoGrub”启动项删除，这样就不用每次进入win10的时候都选择启动项了。当然，C盘中的ubuntu系统iso文件和initrd.lz和vmlinuz.efi文件都可以直接删掉了，节省硬盘空间嘛（毕竟固态硬盘空间比较宝贵）
